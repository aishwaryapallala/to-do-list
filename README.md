## About the Project
This program is a menu-driven To-Do List application that allows users to manage their daily tasks.
The user can add new tasks, view all tasks, and remove tasks by selecting a task number.

## Features
- Add tasks to the list
- View all added tasks
- Remove tasks by task number
- Menu-driven and easy to use
- Handles invalid input safely

## Concepts Used
- Python basics
- Lists
- while loop
- for loop
- if-elif-else conditions
- input() function
- List methods (append, pop)

## How to Run
1. Open the file in Python (IDLE or any editor)
2. Run the program
3. Choose an option from the menu
4. Follow the on-screen instructions

## Learning Outcome
This project helped me understand how to work with lists, loops, and conditional statements in Python.
